# 内蒙古高考报考一键填报--谷歌浏览器插件
## 项目介绍
- **背景**：内蒙古高考报考实时动态排名存在考生在临近提档时间上不能及时更换志愿的问题。为了解决这一问题，我们开发了这款一键填报插件。
- **简介**：该插件允许考生提前预设志愿，在报考中快速切换志愿，提高报考效率，降低因时间不足而导致的报考风险。
